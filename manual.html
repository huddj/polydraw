Polydraw

Polydraw is a polygon and line based drawing tool in a Cartesian coordinate system.

Shapes:

All polygons and lines are organized by the shape system. Shapes are invisible points in space that polygons or other shapes are located in reference to. The 'root' of every drawing is a shape (that should be) located at (0, 0).

Polygons:

Polygons are drawn in reference to a shape. When that shape's relative coordinates change, all the polygons that are 'children' of that shape will move accordingly.

Layers:

Higher layers are displayed above lower layers.

Selection:

The selection menu at the bottom of the screen shows information about the currently selected shape/polygon. It can be used to edit the selected object's properties and to navigate the object tree. Buttons in the selection menu only work if the select tool is active.

Colors:

  • Green: Shape selection
  • Blue: Current active parent shape
  • Red: Polygon selection and creation | polygon point creation and selection
  • Purple: Polyline selection and creation | polyline point creation and selection
  • Orange: Point highlighting

Keys:

For any of these key actions to have effect, the canvas must be focused. Most of the time it will be, but sometimes after editing a text value escape needs to be pressed to focus the canvas. The pointer must also be visible on the canvas.

  • A: "Assign" - If a shape is selected, set it as the current active parent
  • B: "Back" - Select parent of currently selected shape; if a point is selected, select the polygon/polyline the points belongs to
  • C: "Create" - If a polygon/polyline is selected, add a point to the end; if a point is selected add a point before it
  • D: "Delete" - Delete currently selected shape/polygon/polyline/point
  • L: "Line" - Enter polyline creation tool; confirm line after clicking points (creates line relative to current active parent)
  • M: "Move" - Move shape/polygon/polyline/point
  • O: "Outline" - Create a polyline that outlines the current shape and select it
  • P: "Polygon" - Enter polygon creation tool; confirm polygon after clicking points (creates polygon relative to current active parent)
  • R: "Root" - Select root and set it as current active parent
  • S: "Shape" - Create shape (relative to current active parent)
